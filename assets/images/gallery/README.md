# Gallery Images

Place your school gallery images in this folder with the following naming convention:

- 1.jpg
- 2.jpg
- 3.jpg
- 4.jpg
- 5.jpg
- 6.jpg
- 7.jpg
- 8.jpg
- 9.jpg
- 10.jpg

## Image Requirements:
- Format: JPG, JPEG, or PNG
- Recommended size: 800x600 pixels or higher
- Aspect ratio: 4:3 or 16:9 works best

## How it works:
- The carousel displays 3 images at a time
- Auto-slides every 3 seconds
- Pauses when you hover over it
- Navigation buttons and indicators included
- If an image fails to load, a fallback placeholder will be shown

You can add more images by:
1. Adding more image files (11.jpg, 12.jpg, etc.)
2. Updating the galleryImages array in index.html to include the new images
